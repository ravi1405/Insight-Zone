# Insight Zone

Insight Zone is a project aimed at Notes.

## Features

- Feature 1: View/Download Notes Of any university.
- Feature 2: Upload Notes of yours to Help others.
- Feature 3: Get some premium notes of known professors.

## Installation

To use Insight Zone, follow these steps:

1. Clone the repository: `git clone https://github.com/ravi1405/Insight-Zone.git`
2. Install this Libraries : multiselectfield, requests, pillow, magic-bin
   To install this Libraries :
    1. pip install django-multiselectfield
    2. pip install requests
    3. pip install python-magic-bin
    4. pip install pillow
3. Make sure to do migrations before running it.
